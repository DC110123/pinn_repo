## What these plots show

Each `.png` shows the I-V (current-voltage) curve **predicted by the trained
PINN** for a specific panel, plotted against reference points (e.g. the true
max-power point) for a visual sanity check. This is the same style of check
performed by `test_panel()` in the main training pipeline — sweeping voltage
from 0 to `V_oc` and plotting the model's predicted current at each point.

These are qualitative shape checks: the curve should be smooth, monotonically
decreasing (current falls as voltage rises), and should pass near the panel's
known short-circuit current (`I_sc`, at V=0), open-circuit voltage (`V_oc`, at
I=0), and max-power point.
